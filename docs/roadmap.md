# Roadmap — Santa Monica OS

## Sprint 1 — Fundação (concluído nesta execução)

- Projeto Next.js + TypeScript + Tailwind + shadcn-style UI
- Layout, navegação e dashboard executivo
- Módulos demonstrativos (Lavação, Estacionamento, Agenda, Clientes, Financeiro, Marketing,
  Estoque, Compras, Segurança)
- Arquitetura de agentes documentada
- Camada segura de integração JumpPark (sem conexão real ativada)
- Documentação inicial
- Projeto pronto para Vercel

## Sprint 2 — JumpPark

- Configurar variáveis de ambiente reais (fora do repositório)
- Autenticação validada em produção
- Clientes, veículos, ordens de serviço, faturas, financeiro, transações
- Sincronização periódica (somente leitura)

## Sprint 3 — Inteligência

- Zézinho conectado a um modelo real
- Especialistas (Carlos, Bia, Vini, Nina, Eva, Beto, Marta, Radar, Memória)
- Alertas automáticos e recomendações reais
- Histórico de interações

## Sprint 4 — Marketing

- Meta Ads, Instagram, Facebook, Google Business Profile
- Leitura real de campanhas e leads
- Recomendações (manter/pausar/aumentar) com aprovação humana

## Sprint 5 — Estoque e Compras

- Cadastro real de produtos e consumo
- Alertas de reposição
- Integração oficial com Mercado Livre para pesquisa de preços

## Sprint 6 — Segurança

- Módulo Vigia com ponte local segura
- RTSP/ONVIF via NVR
- Alertas de câmera offline e eventos fora do horário
